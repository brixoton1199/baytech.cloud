## Getting Started

### 1. Install Dependencies

```powershell
npm install
```

This will install Vite and other dependencies.

### 2. Development Mode

Start the development server:

```powershell
npm run dev
```

This will:
- Start a local development server (usually at `http://localhost:5173`)
- Watch for file changes and auto-reload
- Provide fast hot module replacement

### 3. Build for Production

Build your website:

```powershell
npm run build
```

This will:
- Compile and optimize your code
- Output files to `baytech.cloud/` directory
- Ready for deployment

### 4. Preview Production Build

Preview the production build locally:

```powershell
npm run preview
```

## Development Workflow

1. **Edit files** in `src/` directory:
   - `src/main.js` - Your JavaScript code
   - `src/style.css` - Your styles

2. **Run dev server**:
   ```powershell
   npm run dev
   ```

3. **View changes** in browser at `http://localhost:5173`

4. **Build for production**:
   ```powershell
   npm run build
   ```

5. **Deploy to Cloudflare**:
   ```powershell
   git add .
   git commit -m "Update website"
   git push origin main
   ```

## Cloudflare Pages Configuration

Update your Cloudflare Pages build settings:

- **Build command**: `npm run build`
- **Build output directory**: `baytech.cloud`
- **Framework preset**: `None` (or `Vite` if available)

## Available Scripts

- `npm run dev` - Start development server

## Adding More Features

### Add a Framework (React, Vue, etc.)

If you want to use React, Vue, or another framework:

```powershell
# For React
npm install react react-dom
npm install -D @vitejs/plugin-react

# For Vue
npm install vue
npm install -D @vitejs/plugin-vue
```

Then update `vite.config.js` accordingly.

### Add More Files

- **Components**: Create in `src/components/`
- **Pages**: Create in `src/pages/`
- **Assets**: Put in `public/` directory

## Troubleshooting

### Build Fails

- Make sure dependencies are installed: `npm install`
- Check for errors in the terminal
- Verify `vite.config.js` is correct

### Changes Not Showing

- Clear browser cache
- Restart dev server: Stop (Ctrl+C) and run `npm run dev` again
- Check browser console for errors
